# Airline UI Glossary

This is a practical review reference, not an official airline dictionary. Confirm each carrier's product model, regional usage, and operational definitions before changing a status or policy term.

| Term | Usage note |
|---|---|
| Booking | A reservation or itinerary record; do not use it when the UI specifically means a ticket or completed purchase. |
| My trips | Common customer-facing navigation for upcoming and past travel; usually broader and friendlier than “My bookings.” |
| Manage booking | Use for changing or viewing an existing reservation; confirm whether the product uses “booking” or “trip” consistently. |
| Flight status | Use for the operational state or timing of a flight, not the state of a passenger's booking. |
| Check-in | Use as a noun or modifier; use “Check in” as the verb. |
| Boarding pass | The document or digital credential used for boarding; it is not the same as a booking confirmation. |
| Passenger details | Information about a traveler; use a narrower label when only identity or contact data is requested. |
| Seat selection | The process or section for choosing seats; confirm whether selection is free, paid, or restricted. |
| Extra baggage | General customer-facing term for baggage beyond the included allowance; do not imply it was already purchased. |
| Prepaid baggage | Baggage allowance purchased before airport handling; carrier definitions and cutoff times may vary. |
| Baggage allowance | The amount or number of bags included or permitted; specify weight, pieces, route, or fare rules when relevant. |
| Special assistance | Broad support category for passengers who need additional help; avoid using it as a substitute for a specific service. |
| Wheelchair assistance | A specific assistance request; clarify airport stages or service level when the product distinguishes them. |
| Scheduled | Refers to the published plan or scheduled time; do not use it for an updated estimate. |
| Estimated | Refers to a current forecast, commonly for departure or arrival time; label the relevant time. |
| Delayed | The flight is expected to operate later than scheduled; do not imply cancellation. |
| Cancelled | The flight will not operate as scheduled; use the product's regional spelling consistently (`Cancelled` or `Canceled`). |
| Departed | The flight has left; confirm whether the data source means gate departure or takeoff if the distinction matters. |
| Arrived | The flight has reached its arrival state; confirm whether the system means landing or gate arrival when needed. |
| Boarding | Passengers are currently being admitted for the flight; distinguish from a future boarding time. |
| Check-in open | Check-in is currently available; include channel or time restrictions when material. |
| Check-in closed | Check-in is no longer available through the stated channel; do not imply the flight itself is closed or cancelled. |
| Member | A participant in a loyalty program or account group; name the program or group when “member” is ambiguous. |
| Family account | An account or grouping feature for related members; do not imply pooled ownership unless the rules support it. |
| Invite member | Action for inviting a person to a group or account; state eligibility or approval requirements when relevant. |
| Earn miles | Accumulate loyalty currency under program rules; avoid implying that every purchase or flight qualifies. |
| Redeem miles | Use loyalty currency for an eligible reward or purchase; distinguish from transferring or converting miles. |
| Mileage balance | The number of loyalty miles recorded in the account; clarify available, pending, or expiring miles when the distinction exists. |

## Review cautions

- Keep booking status, ticket status, payment status, and flight status separate.
- Do not use “ticket,” “booking,” and “trip” as automatic synonyms.
- Status definitions may depend on the operational data source; request confirmation when a wording change could alter the represented state.
- Spellings such as `Cancelled` and `Canceled` depend on the selected English locale.
