# Slide 截图翻译示例

## 输入

用户请求：把这一页 PPT 翻译成英文。

截图中可见文字：

```text
STEP 02 · 定位
产品性格

首页应该是一个企业资金工作台，
而不是一个零售门户。

改版前
零售门户
入口太多，以个人银行为中心，重点是“我能做什么？”
- 导航优先
- 服务优先
- 功能集合

改版后
企业资金工作台
资产、任务和数据优先，帮助客户更快决策。
- 一眼看到资产状态
- 优先展示任务和现金流
- 更清晰的跨境金融上下文

底部操作提示：
← Navigate · B Static · ESC Index
```

## 输出

### 可直接替换为

**STEP 02 · POSITIONING**
**PRODUCT CHARACTER**

The homepage should be a corporate treasury workspace,
not a retail portal.

**BEFORE**
**Retail Portal**

Too many entry points, built around personal banking and focused on “What can I do?”

- Navigation-first
- Service-first
- A collection of features

**AFTER**
**Corporate Treasury Workspace**

Assets, tasks, and data come first, helping clients make faster decisions.

- Asset status at a glance
- Tasks and cash flows prioritized
- Clearer cross-border finance context

Footer operation tips:
`← Navigate · B Static · ESC Index`

### 翻译说明

- “企业资金工作台”不建议直译为 `enterprise fund workbench`；`Corporate Treasury Workspace` 更符合金融产品方案语境。
- “零售门户”使用 `Retail Portal`，与企业资金工作台形成清晰对比。
- “产品性格”可译为 `Product Character`；如果页面强调商业定位，也可以使用 `Product Positioning`。

### 需确认内容

如果这是正式对外提案，需要确认 `Corporate Treasury Workspace` 是否与客户内部产品定位一致。
